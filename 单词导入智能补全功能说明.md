# 单词导入智能补全功能说明

## ✨ 功能概述

单词导入功能已优化，现在支持智能自动补全：
- **手工录入**：只需输入单词，系统自动查询释义、音标、例句
- **Excel/CSV导入**：只要有单词列，其他缺失信息自动补全

---

## 📝 使用方式

### 方式一：手工录入单词

1. 进入"单词导入"页面
2. 点击"✍️ 手工录入"卡片
3. **只需输入单词**（例如：abandon）
4. 点击"🔍 查询"按钮或按回车键
5. 系统自动显示查询结果：
   - 音标
   - 中文释义
   - 英文释义
   - 例句
6. 可以手动修改查询结果（如果需要）
7. 点击"添加到列表"
8. 重复添加多个单词后，点击"批量导入"

**批量文本输入**：
- 支持格式：`单词 | 释义 | 音标 | 例句`
- 示例：
  ```
  abandon | 放弃 | /əˈbændən/ | He abandoned his wife.
  ability | 能力
  about | 关于 | /əˈbaʊt/
  ```
- 点击"解析文本"后自动查询补全缺失信息

### 方式二：Excel/CSV导入

1. 准备Excel或CSV文件
2. **必填列**：单词（或 Word）
3. **选填列**（可留空，系统自动补全）：
   - 中文释义
   - 英文释义
   - 例句
   - 音标
4. 上传文件后，系统自动解析
5. 预览解析结果
6. 点击"确认导入"
7. **后端自动查询并补全所有缺失信息**

**示例Excel格式**：
| 单词 | 中文释义 | 英文释义 | 例句 | 音标 |
|------|---------|---------|------|------|
| apple | 苹果 | | | /ˈæpl/ |
| book | | A written work | | |
| cat | 猫 | | The cat is sleeping. | |

导入后，所有空白字段都会自动从网络查询补充。

## 🔧 技术实现

### 前端（Import.vue）

**手工录入流程**：
```
用户输入单词 → 点击查询 → 调用 /api/translate → 显示结果 → 用户确认 → 添加到列表 → 批量导入
```

**关键改进**：
1. 简化表单：只保留单词输入框
2. 添加查询按钮：实时查询单词信息
3. 查询结果展示：绿色卡片显示自动查询的信息
4. 支持手动修改：用户可以调整查询结果

### 后端（wordbank.js）

**批量导入流程**：
```
接收单词列表 → enrichWordsData() → 遍历每个单词 → 检查缺失字段 → 调用翻译API → 补全数据 → 存入数据库
```

**关键改进**：
1. 始终自动补全：移除autoFill参数，所有批量导入都自动补全
2. 智能判断：只查询缺少 definition_cn、phonetic 或 definition_en 的单词
3. 保留已有数据：不会覆盖用户已提供的信息
4. 详细日志：显示每个单词的查询状态

**核心函数**：
```javascript
async function enrichWordsData(words) {
  // 遍历单词列表
  for (const word of words) {
    // 检查是否需要查询
    const needsQuery = !word.definition_cn || !word.phonetic || !word.definition_en
    
    if (needsQuery && word.word) {
      // 调用翻译API补全
      const translateResult = await callBaiduTranslate(word.word)
      // 只填充缺失字段
    }
  }
}
```

### 翻译API（translate.js）

- 端点：`POST /api/translate`
- 输入：`{ word: "abandon" }`
- 输出：
  ```json
  {
    "word": "abandon",
    "phonetic": "/əˈbændən/",
    "definition_cn": "放弃；抛弃",
    "definition_en": "to leave someone or something permanently",
    "example": "He abandoned his wife and children."
  }
  ```

**注意**：当前使用模拟数据（未配置百度翻译API），实际使用时需要配置：
```env
BAIDU_APP_ID=your_app_id
BAIDU_APP_KEY=your_app_key
```

## 🎯 用户体验优化

1. **最小化输入**：用户只需输入单词，其他全自动
2. **实时反馈**：查询时显示"查询中..."状态
3. **可修改**：查询结果可以手动调整
4. **批量处理**：支持一次导入多个单词
5. **智能跳过**：已有完整信息的单词不会重复查询
6. **错误容忍**：查询失败不影响导入，使用已有数据

## 🐛 注意事项

1. **网络依赖**：自动补全需要网络连接
2. **API限制**：百度翻译API有调用频率限制（免费版QPS=1）
3. **模拟数据**：未配置API时使用模拟数据（仅支持少量单词）
4. **查询时间**：大量单词导入可能需要较长时间（每个单词1-2秒）
5. **部分失败**：个别单词查询失败不影响整体导入
