# 百词斩风格设计说明

> **项目名称：** 中小学生背单词小程序  
> **设计风格：** 百词斩风格 + 自定义词库强化  
> **版本：** v2.0  
> **设计日期：** 2026-04-11

---

## 🎨 设计理念

### 参考百词斩的核心元素
1. **卡片式学习** - 大圆角卡片，阴影效果，层次分明
2. **渐变背景** - 活力配色，视觉冲击力强
3. **图文结合** - 单词配图，增强记忆
4. **游戏化交互** - 滑动操作，动画丰富，即时反馈
5. **进度可视化** - 清晰的进度条和统计数据

### 核心差异化功能
✨ **用户自定义词库** - 百词斩没有的灵活功能  
✨ **批量导入单词** - 拍照OCR、Excel批量导入  
✨ **智能词典补充** - 自动补充释义、例句、发音  
✨ **艾宾浩斯复习** - 科学记忆算法  

---

## 🎯 设计对比

### 百词斩 vs 我们的设计

| 特性 | 百词斩 | 我们的设计 | 说明 |
|------|--------|-----------|------|
| 词库管理 | 固定词库，不可自定义 | ✅ 用户可创建任意词库 | 核心差异 |
| 导入方式 | 不支持 | ✅ 拍照/图片/Excel | 批量添加 |
| 学习界面 | 卡片式，图文结合 | ✅ 卡片式，配图+渐变 | 参考+优化 |
| 交互方式 | 滑动选择 | ✅ 滑动+点击 | 更丰富 |
| 进度展示 | 简洁进度条 | ✅ 点阵进度+百分比 | 更直观 |
| 色彩方案 | 清新明亮 | ✅ 渐变背景+活力配色 | 更现代 |

```

```

### 2. 圆角规范
```css
卡片: 16-20px
按钮: 24px (完全圆角)
单词卡片: 24px
选项按钮: 16px
头像: 50% (圆形)
```

### 3. 阴影效果
```css
卡片阴影: 0 4px 12px rgba(0,0,0,0.08)
按钮阴影: 0 4px 12px rgba(0,180,216,0.3)
单词卡片: 0 8px 24px rgba(0,0,0,0.15)
```

### 4. 字体大小
```css
页面标题: 24px / 700 weight
卡片标题: 18px / 700 weight
正文: 14-16px / 400-500 weight
次要文字: 12-13px / 400 weight
学习单词: 36px / 700 weight
统计数据: 28-48px / 700 weight
```

- 本周学习进度大数字展示

✨ 连续学习天数徽章

### 2. 词库管理页（核心强化）

**设计亮点：**

**百词斩元素：**

**差异化功能：**
✨ 用户可创建任意词库

**百词斩元素：**

- 每种方式都有图标、标题、说明

**核心功能：**

✨ 手动输入（灵活）  

**用户体验：**

- 图标直观，一目了然
- 实时统计，反馈及时

## 🎮 交互设计规范

### 1. 页面切换
- 底部导航栏切换，带active状态
- 页面切换平滑过渡（300ms）
- 切换后自动滚动到顶部

### 2. 按钮交互
```css
默认状态: 渐变背景 + 阴影
按下状态: transform: scale(0.98)
          box-shadow 减小
```

### 3. 卡片交互
```css
默认状态: 阴影 0 4px 12px
按下状态: transform: translateY(-2px)
          阴影 0 6px 16px
```

### 4. 选项反馈
```css
正确: border-color: #06D6A0
      background: #f0fff4
错误: border-color: #EF476F
      background: #fff0f3
```

### 5. 滑动操作（学习页面）
- 左滑：标记"不认识"
- 右滑：标记"认识"
- 提示文字引导用户

## 🚀 开发建议

### 1. CSS框架推荐
- **Tailwind CSS** - 快速开发，实用类
- **Vant 4** - 移动端组件库（可选）
- **自定义CSS** - 更灵活，推荐

### 2. 动画库推荐
- **Animate.css** - 预设动画
- **Framer Motion** - React项目
- **自定义CSS动画** - 更轻量

### 3. 图标方案
- **Emoji** - 原型阶段，简单直观
- **Iconfont** - 生产环境，自定义图标
- **SVG图标** - 高质量，可缩放

### 4. 渐变生成工具
- **CSS Gradient** - https://cssgradient.io/
- **Gradient Hunt** - https://gradienthunt.com/
- **WebGradients** - https://webgradients.com/

## 📝 与百词斩的区别总结

### 视觉层面
| 维度 | 百词斩 | 我们的设计 |
|------|--------|-----------|
| 背景 | 纯色或简单渐变 | 多色渐变，更丰富 |
| 卡片 | 圆角12px | 圆角16-24px，更大 |
| 阴影 | 较浅 | 更明显，层次更强 |
| 按钮 | 标准圆角 | 完全圆角，更现代 |

### 功能层面
| 功能 | 百词斩 | 我们的设计 |
|------|--------|-----------|
| 词库 | 固定，不可修改 | ✅ 自由创建和管理 |
| 导入 | 不支持 | ✅ 拍照/图片/Excel |
| 批量添加 | 不支持 | ✅ 多种批量导入方式 |
| 自定义 | 有限 | ✅ 完全自定义 |

### 交互层面
| 交互 | 百词斩 | 我们的设计 |
|------|--------|-----------|
| 学习 | 滑动+点击 | 滑动+点击+更丰富反馈 |
| 进度 | 简单进度条 | 点阵+进度条+百分比 |
| 统计 | 基础统计 | 详细统计+可视化 |
