# UI原型使用说明

> **项目名称：** 中小学生背单词小程序  
> **原型版本：** v1.0  
> **创建日期：** 2026-04-11

---

## 📱 原型概览

这是一个**完整的HTML交互原型**，展示了背单词小程序的所有核心页面和交互流程。

### 包含页面
1. ✅ **首页** - 学习仪表盘
2. ✅ **词库管理页** - 查看和切换词库
3. ✅ **学习页面** - 看词识意模式
4. ✅ **复习页面** - 艾宾浩斯复习
5. ✅ **导入词库页** - 图片/Excel导入
6. ✅ **个人中心** - 设置和统计

---

## 🚀 如何查看原型

### 方法1：直接打开（推荐）
1. 找到文件：`design/ui-prototype.html`
2. 双击用浏览器打开
3. 按 `F12` 打开开发者工具
4. 切换到**移动端视图**（Ctrl+Shift+M）
5. 选择 **iPhone 12/13** 或 **Pixel 5** 预设

### 方法2：本地服务器
```bash
# 进入design目录
cd design

# 使用Python启动本地服务器
python -m http.server 8080

# 浏览器访问
http://localhost:8080/ui-prototype.html
```

### 方法3：VS Code Live Server
1. 在VS Code中打开 `ui-prototype.html`
2. 右键选择 "Open with Live Server"
3. 自动在浏览器中打开

## 🎮 交互说明

### 底部导航栏
- 🏠 **首页** - 查看学习进度和今日任务
- 📚 **词库** - 管理和切换词库
- 🔄 **复习** - 进行艾宾浩斯复习
- 👤 **我的** - 个人中心和设置

### 页面跳转
- 点击首页的 **"开始学习"** → 进入学习页面
- 点击首页的快捷入口 → 跳转到对应页面
- 点击词库管理的 **"导入新词库"** → 进入导入页面
- 所有页面都可通过底部导航切换

### 学习页面交互
- 点击选项会显示**正确/错误**状态
- 正确答案显示绿色，错误答案显示红色
- 选择后会弹出提示框

### 1. 视觉设计
- ✅ **简洁明快** - 大量留白，信息层次清晰

- ✅ **操作流畅** - 页面切换无卡顿
- ✅ **反馈及时** - 按钮点击有视觉反馈
- ✅ **引导清晰** - 进度条、统计数字一目了然

### 3. 用户体验
- ✅ **学习路径明确** - 首页→学习→复习，流程自然
- ✅ **进度可视化** - 每个环节都有进度展示
- ✅ **快捷入口** - 常用功能一键可达

## 📋 页面详细说明

### 1. 首页（学习仪表盘）
**功能：** 展示学习概况，快速开始学习

**核心元素：**
- 学习天数、已学单词统计
- 今日学习任务（新单词+复习）
- 本周学习进度环
- 快捷入口（词库、导入、复习）

**设计意图：** 让用户一目了然看到学习进度，激励继续学习

### 2. 词库管理页
**功能：** 查看和管理所有词库

**核心元素：**
- 搜索框（快速查找词库）
- 词库列表（显示进度）
- 导入/创建按钮

**设计意图：** 清晰的词库管理，方便切换和添加

### 3. 学习页面（看词识意）
**功能：** 单词学习核心界面

**核心元素：**
- 单词显示（大字+音标）
- 发音按钮
- 四个选项
- 进度和计时

**设计意图：** 专注学习，减少干扰，操作简单

### 4. 复习页面
**功能：** 根据遗忘曲线安排复习

**核心元素：**
- 复习统计（待复习/已复习/正确率）
- 复习进度条
- 复习计划说明

**设计意图：** 科学复习，可视化进度

### 5. 导入词库页
**功能：** 通过多种方式导入单词

**核心元素：**
- 图片导入选项
- Excel导入选项
- 模板下载
- 导入历史

**设计意图：** 降低导入门槛，提供多种方式

### 6. 个人中心
**功能：** 用户设置和学习统计

**核心元素：**
- 头像和学习天数
- 学习统计数据
- 设置菜单

**设计意图：** 个性化设置，数据驱动
