# 🧪 导入功能测试指南

## ✅ 已修复的问题

### 问题描述
导入时报错：`NOT NULL constraint failed: words.bank_id`

### 原因分析
- `bankId` 从路由参数 `route.query.bankId` 获取
- 如果参数不存在或不是有效数字，`parseInt()` 会返回 `NaN`
- `NaN` 传递给数据库导致 `bank_id` 字段为 NULL

### 修复方案
1. **前端验证** - 导入前检查 `bankId` 是否存在且有效
2. **后端验证** - 检查 `bankId` 是否为有效数字
3. **详细日志** - 记录每个导入步骤，方便调试
4. **数据验证** - 验证每个单词都有必需的 `word` 字段

## 📝 测试步骤

### 第一步：创建词库

1. 访问 http://localhost:3000
2. 点击底部导航栏的 **"词库"**
3. 点击 **"+ 创建词库"** 按钮
4. 输入词库名称，例如：`测试词库01`
5. 点击确定

### 第二步：进入导入页面

1. 在词库列表中，找到刚创建的词库
2. 点击 **"导入单词"** 按钮
3. 确认页面顶部显示：**选择词库：测试词库01**

### 第三步：选择文件

1. 点击 **"Excel/CSV导入"** 卡片
2. 选择项目中的测试文件：
   - `测试导入-CSV格式.csv`
   - 或 `词库导入模板.csv`

### 第四步：预览数据

1. 系统会解析文件并显示预览
2. 检查预览中的单词数据是否正确
3. 确认单词数量

### 第五步：确认导入

1. 点击 **"确认导入"** 按钮
2. 查看导入进度
3. 等待导入完成

### 第六步：验证结果

1. 导入成功后，点击 **"返回词库"**
2. 查看词库的单词数量是否更新
3. 可以点击查看词库详情，确认所有单词都已导入

##  常见错误及解决

### 错误1：未选择词库
**提示：** `错误：未选择词库，请返回重新选择`

**原因：** 没有从词库页面进入导入页面

**解决：** 
1. 返回词库列表
2. 点击某个词库的"导入单词"按钮
3. 确保URL中有 `?bankId=xxx` 参数

### 错误2：词库ID无效
**提示：** `错误：词库ID无效`

**原因：** URL参数格式错误

**解决：** 刷新页面或重新从词库列表进入

### 错误3：单词已存在
**提示：** `单词已存在`

**原因：** 词库中已有相同单词

**解决：** 
- 系统会自动跳过重复单词
- 或使用新的词库导入

### 错误4：解析文件失败
**提示：** `解析文件失败：xxx`

**原因：** 文件格式不正确

**解决：**
- 检查CSV格式是否正确
- 确保第一行是列名
- 使用项目提供的测试文件
