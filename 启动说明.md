## 📝 使用流程

### 1. 创建词库
1. 点击底部 **"词库"**
2. 点击 **"+ 创建词库"**
3. 输入名称，如"三年级英语"
4. 点击确定

### 2. 导入单词
1. 点击词库 → **"导入单词"**
2. 点击 **"Excel导入"**
3. 选择Excel文件（或使用项目中的 `词库导入模板.csv`）
4. 预览数据
5. 点击 **"确认导入"**

### 3. 查看结果
返回词库列表，可以看到单词数量已更新！

## 🐛 常见问题

### 1. 双击start.bat闪退
- 右键点击 `start.bat` → "编辑"
- 查看是否有错误信息
- 尝试使用方法二（命令行启动）

### 2. 端口5000被占用
```bash
# 修改端口
# 编辑 backend/server.js，将 PORT = 5000 改为其他端口
```

### 3. npm install 失败
```bash
# 清除缓存
npm cache clean --force
# 重新安装
npm install
```

### 4. 数据库错误
```bash
# 删除数据库重新创建
rm -rf data
# 重启服务器会自动创建
```

### 5. 创建词库按钮没反应
**原因：** 页面缓存或Vant组件未加载

**解决：**
- 刷新页面（Ctrl+F5）
- 清除浏览器缓存（Ctrl+Shift+Delete）
- 检查浏览器控制台是否有错误（F12）

### 6. 导入时提示“未选择词库”
**原因：** 没有从词库页面进入导入页面

**解决：**
1. 点击“管理词库”创建词库
2. 在词库列表中点击“导入单词”
3. 系统会自动带上词库ID

### 7. 导入失败
**查看日志：**
- 前端：按F12打开浏览器控制台
- 后端：查看启动服务器的终端窗口

**常见错误：**
- `NOT NULL constraint failed`: 未选择词库
- `单词已存在`: 词库中已有相同单词（会自动跳过）
- `词库ID无效`: URL参数错误
