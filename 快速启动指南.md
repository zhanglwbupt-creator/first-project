# 🚀 快速启动指南

## 第一步：安装依赖

在项目目录下运行：

```bash
npm install
```

这会安装所有必需的前端和后端依赖。

---

## 第二步：启动后端服务器

```bash
npm run server
```

您会看到：
```
🚀 服务器已启动: http://localhost:5000
📊 API文档: http://localhost:5000/api/health
```

**保持这个终端运行！**

---

## 第三步：启动前端开发服务器

**打开新的终端**，在项目目录下运行：

```bash
npm run dev
```

您会看到：
```
VITE v5.0.0  ready in xxx ms

➜  Local:   http://localhost:3000/
```

---

## 第四步：访问应用

在浏览器中打开：**http://localhost:3000**

您应该能看到背单词应用的首页！

---

## 📝 使用流程

### 1. 创建词库

1. 点击底部导航栏的 **"词库"**
2. 点击 **"+ 创建词库"** 按钮
3. 输入词库名称，例如："三年级英语"
4. 点击确定

### 2. 导入单词

#### 方式一：使用模板文件

1. 找到项目中的模板文件：
   - `词库导入模板.csv` - CSV格式
   - 或在应用中下载Excel模板
2. 用Excel打开并编辑，或直接使用
3. 在应用中点击 **“词库”** → 选择词库 → **“导入单词”**
4. 点击 **“Excel/CSV导入”**
5. 选择Excel或CSV文件
6. 预览数据，点击 **“确认导入”**

#### 方式二：创建自己的Excel或CSV

**Excel格式：**

创建一个Excel文件（.xlsx或.xls），包含以下列：

| 单词 | 中文释义 | 英文释义 | 例句 | 音标 |
|------|---------|---------|------|------|
| apple | 苹果 | A round fruit... | I eat an apple... | /ˈæpl/ |
| book | 书 | A written work... | I am reading... | /bʊk/ |

**CSV格式：**

创建一个CSV文件（.csv），内容如下：

```csv
单词,中文释义,英文释义,例句,音标
apple,苹果,A round fruit...,I eat an apple...,/ˈæpl/
book,书,A written work...,I am reading...,/bʊk/
```

**注意：** 
- 只有“单词”列是必填的，其他列可以为空，系统会自动查询补充！
- CSV文件建议使用UTF-8编码，避免中文乱码

### 3. 查看导入结果

1. 返回词库列表
2. 可以看到词库的单词数量已更新
3. 点击词库可以查看导入的所有单词

## 🔧 功能说明

### ✅ 已实现功能

1. **词库管理**
   - 创建词库
   - 查看词库列表
   - 删除词库
   - 显示单词数量

2. **Excel/CSV导入**
   - 支持 .xlsx、.xls 和 .csv 格式
   - 智能列名识别
   - 批量导入
   - 导入预览
   - 进度显示

3. **自动词典查询**
   - 如果Excel中缺少释义，系统自动查询
   - 查询内容包括：
     - 中文释义
     - 英文释义
     - 例句
     - 音标
   - 内置模拟数据（用于测试）

## 🐛 常见问题

### 1. npm install 失败
```bash
# 清除缓存重试
npm cache clean --force
npm install
```

### 2. 端口被占用
```bash
# 修改后端端口
# 编辑 backend/server.js，修改 PORT = 5000 为其他端口

# 修改前端端口
# 编辑 vite.config.js，修改 port: 3000 为其他端口
```

### 3. 数据库错误
```bash
# 删除数据库重新创建
rm -rf data
# 重新启动服务器，会自动创建
```

### 4. Excel导入失败
- 确保Excel第一行是列名
- 确保有"单词"列
- 可以尝试使用项目中的 `词库导入模板.csv`
