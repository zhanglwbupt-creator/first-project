# 百度翻译API配置指南

## 📋 快速开始

### 第一步：注册百度翻译API

1. **访问百度翻译开放平台**
   - 网址：https://fanyi-api.baidu.com/
   - 点击"登录/注册"

2. **注册账号**
   - 使用百度账号登录（如果没有，先注册百度账号）
   - 完成实名认证（需要手机号验证）

3. **开通翻译API服务**
   - 登录后，进入"管理控制台"
   - 点击"开通服务"
   - 选择"通用翻译API"（免费版）
   - 阅读并同意服务协议

4. **获取API密钥**
   - 在控制台找到"开发者信息"
   - 查看您的 **APP ID** 和 **密钥**
   - 复制这两个值

### 第二步：配置项目

1. **打开项目根目录的 `.env` 文件**
   ```
   d:/WORK/AI CODING TEST/wordbook_20260405/.env
   ```

2. **修改以下两行**
   ```env
   BAIDU_APP_ID=your_app_id_here        # 替换为您的APP ID
   BAIDU_APP_KEY=your_app_key_here      # 替换为您的密钥
   ```

   **示例**：
   ```env
   BAIDU_APP_ID=20240101001234567
   BAIDU_APP_KEY=AbCdEfGhIjKlMnOpQrSt
   ```

3. **保存文件**

### 第三步：重启服务器

```bash
# 停止当前服务器（Ctrl+C）
# 重新启动
npm run server
```

### 第四步：验证配置

查看服务器控制台输出：
- ✅ **成功**：不再显示"⚠️ 未配置百度翻译API，使用模拟数据"
- ❌ **失败**：仍然显示警告信息，检查`.env`文件配置

## 🎁 免费版额度说明

百度翻译API免费版：
- **QPS（每秒请求数）**：1
- **每月字符数**：200万字符
- **适合场景**：个人学习、小规模应用

**注意事项**：
- QPS=1意味着不能并发查询，需要逐个单词查询
- 导入大量单词时会有延迟（每个单词1-2秒）
- 建议配置后测试少量单词

## ❌ 常见问题

### Q1: 配置后仍然显示模拟数据？

**解决方法**：
1. 确认`.env`文件在项目根目录（与package.json同级）
2. 确认变量名正确：`BAIDU_APP_ID` 和 `BAIDU_APP_KEY`
3. 重启服务器（必须重启才能加载新配置）
4. 检查是否有空格或特殊字符

### Q2: API调用失败？

**可能原因**：
- APP ID或密钥错误
- 服务未开通
- 账号未实名认证
- 超出免费额度

**检查方法**：
查看服务器控制台错误信息，或访问：
https://fanyi-api.baidu.com/api/trans/vip/translate?q=test&from=en&to=zh&appid=YOUR_APP_ID&salt=123&sign=YOUR_SIGN

### Q3: 导入速度很慢？

**原因**：免费版QPS=1，每个单词需要串行查询

**解决方案**：
- 耐心等待（100个单词约2-3分钟）
- 或升级到付费版（QPS更高）
- 或在Excel中预先填写部分释义

BAIDU_APP_ID=20240101001234567

**安全提示**：
- `.env`文件不应提交到Git仓库
- 已添加到`.gitignore`中
- 不要在代码中硬编码密钥

##  升级建议

如果免费版不够用，可以考虑：

1. **百度翻译付费版**
   - 标准版：QPS=10，适合中小型应用
   - 高级版：QPS=100，适合大型应用

2. **多API源备用**
   - 有道翻译API
   - 金山词霸API
   - 自建词典缓存

3. **本地缓存优化**
   - 查询过的单词保存到本地
   - 避免重复查询相同单词
