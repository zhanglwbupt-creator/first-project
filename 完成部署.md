## 🐛 如果遇到问题

### 问题1：npm命令找不到

**解决：** 使用完整路径
```bash
/usr/bin/node /usr/lib/node_modules/npm/bin/npm-cli.js install -g pm2
```

### 问题2：pm2命令找不到

**解决：** 设置PATH
```bash
export PATH=$HOME/.local/bin:$PATH
```

### 问题3：服务启动失败

**查看日志：**
```bash
pm2 logs wordbook --lines 100
```

**常见错误：**
- 端口被占用：`pm2 delete wordbook` 然后重新启动
- 依赖缺失：`cd ~/wordbook && npm install`

### 问题4：无法访问网站

**检查清单：**
1. ✅ 服务是否运行：`pm2 status`
2. ✅ 端口是否监听：`netstat -tlnp | grep 5000`
3. ✅ 防火墙是否开放：`sudo ufw status`
4. ✅ **云安全组是否开放5000端口**（最容易忘记！）
