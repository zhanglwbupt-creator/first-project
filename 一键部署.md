# 🚀 一键部署命令

## 最简单的部署方式

### 第1步：SSH登录云主机

打开PowerShell或终端，执行：

```bash
ssh newuser@82.156.202.245
```

输入密码：`YourPassword123!`

### 第2步：复制粘贴以下整条命令（一键完成所有部署）

```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -S bash - <<< 'YourPassword123!' && sudo -S apt-get install -y nodejs <<< 'YourPassword123!' && mkdir -p ~/wordbook && cd ~/wordbook && tar -xzf ~/wordbook-deploy.tar.gz && npm install --production && npm install -g pm2 && sudo -S ufw allow 5000/tcp <<< 'YourPassword123!' && pm2 start backend/server.js --name wordbook && pm2 save && echo "部署完成！访问 http://82.156.202.245:5000"
```

**或者直接复制这行（已格式化）：**

```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -S bash - <<< 'YourPassword123!'
sudo -S apt-get install -y nodejs <<< 'YourPassword123!'
mkdir -p ~/wordbook
cd ~/wordbook
tar -xzf ~/wordbook-deploy.tar.gz
npm install --production
npm install -g pm2
sudo -S ufw allow 5000/tcp <<< 'YourPassword123!'
pm2 start backend/server.js --name wordbook
pm2 save
echo "✅ 部署完成！访问 http://82.156.202.245:5000"
```

### 第3步：等待部署完成

整个过程约2-5分钟，完成后访问：
```
http://82.156.202.245:5000
```

## 📋 分步执行（如果上面的一行命令有问题）

登录SSH后，依次执行：

```bash
# 1. 安装Node.js（会提示输入密码）
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo bash -
sudo apt-get install -y nodejs

# 2. 创建目录并解压
mkdir -p ~/wordbook
cd ~/wordbook
tar -xzf ~/wordbook-deploy.tar.gz

# 3. 安装依赖
npm install --production

# 4. 安装PM2
npm install -g pm2

# 5. 开放防火墙
sudo ufw allow 5000/tcp

# 6. 启动服务
pm2 start backend/server.js --name wordbook
pm2 save

# 7. 验证
pm2 status
```

## ⚠️ 重要：开放云安全组端口

登录云主机控制台，确保：
- 添加入站规则：TCP 5000端口
- 授权对象：0.0.0.0/0

## 🎯 验证部署

```bash
# 检查服务
pm2 status

# 检查端口
netstat -tlnp | grep 5000

# 测试API
curl http://localhost:5000/api/health
```

## 🐛 常见问题

### npm安装很慢？
```bash
npm config set registry https://registry.npmmirror.com
npm install --production
```

### 无法访问网站？
1. 检查服务：`pm2 status`
2. 检查防火墙：`sudo ufw status`
3. **检查云安全组**（最容易忘记）
