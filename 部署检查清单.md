### 方式一：自动部署（最简单）

```bash
# 双击运行

```

**注意**：需要云主机已安装Node.js

### 方式二：手动部署（推荐，更可控）

#### 第1步：上传文件到云主机 ⬜
```powershell
scp wordbook-deploy.tar.gz root@82.156.202.245:/root/
```
**输入云主机密码**

#### 第2步：SSH连接云主机 ⬜
```bash
ssh root@82.156.202.245
```
**输入云主机密码**

#### 第3步：检查Node.js环境 ⬜
```bash
node -v
npm -v
```

**如果没有安装**，选择对应系统：

**CentOS/阿里云Linux：**
```bash
curl -fsSL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs
```

**Ubuntu/Debian：**
```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt-get install -y nodejs
```

#### 第4步：解压并安装 ⬜
```bash
# 创建目录
mkdir -p /root/wordbook

# 解压
tar -xzf /root/wordbook-deploy.tar.gz -C /root/wordbook

# 进入目录
cd /root/wordbook

# 安装依赖
npm install --production

# 安装PM2
npm install -g pm2
```

#### 第5步：开放端口 ⬜

**系统防火墙：**
```bash
# CentOS
firewall-cmd --permanent --add-port=5000/tcp
firewall-cmd --reload

# Ubuntu
ufw allow 5000/tcp
```

**云主机安全组（重要！）：**
- 登录云控制台（阿里云/腾讯云/华为云）
- 找到"安全组规则"
- 添加入站规则：
  - 端口：5000
  - 协议：TCP
  - 授权对象：0.0.0.0/0

#### 第6步：启动服务 ⬜
```bash
pm2 start backend/server.js --name wordbook
pm2 save
pm2 startup
```

#### 第7步：验证访问 ⬜
浏览器打开：http://82.156.202.245:5000

应该看到登录页面！✅

## 🐛 常见问题

### Q1: SSH连接被拒绝
**解决**：
- 检查云主机是否运行中
- 检查安全组是否开放22端口
- 检查密码是否正确

### Q2: 无法访问网站
**排查**：
1. `pm2 status` - 服务是否运行
2. `netstat -tlnp | grep 5000` - 端口是否监听
3. 防火墙是否开放5000端口
4. **云安全组是否开放5000端口**（最容易忘记！）

### Q3: Node.js版本太低
**解决**：
```bash
# 卸载旧版本
yum remove -y nodejs

# 安装Node.js 18
curl -fsSL https://rpm.nodesource.com/setup_18.x | bash -
yum install -y nodejs
```

### Q4: npm install 很慢
**解决**：使用淘宝镜像
```bash
npm config set registry https://registry.npmmirror.com
npm install --production
```
