# 学习进度全局更新机制

## 📊 功能概述

实现了学习进度的全局实时更新机制，确保用户完成学习后，程序中所有需要显示统计的地方都能自动更新。

## 💡 技术实现

### 前端事件机制

#### 1. 触发事件（学习完成后）
```javascript
// StudyReport.vue
window.dispatchEvent(new CustomEvent('studyCompleted', { 
  detail: { bankId: parsed.bankId } 
}))
```

#### 2. 监听事件（各页面）
```javascript
// Home.vue, WordBank.vue, WordBankDetail.vue, StudyMode.vue
onMounted(() => {
  // 初始加载
  loadStats()
  
  // 监听学习完成事件
  window.addEventListener('studyCompleted', loadStats)
})

onUnmounted(() => {
  // 清理事件监听器
  window.removeEventListener('studyCompleted', loadStats)
})
```

---

### 后端计算逻辑

#### 进度百分比计算
```javascript
const progress = totalWords > 0 
  ? Math.round((stats.totalLearned / totalWords) * 100) 
  : 0
```

#### 正确率计算
```javascript
const accuracy = accuracyData.total > 0 
  ? Math.round((accuracyData.correct_count / accuracyData.total) * 100) 
  : 0
```

## 📈 数据统计维度

### 1. 单词维度
- **总单词数** - 词库中所有单词
- **已学习** - 至少学习过一次的单词
- **已掌握** - 状态为2（完全掌握）的单词
- **待学习** - 从未学习过的单词

### 2. 时间维度
- **今日已学** - 今天学习的单词数
- **累计学习** - 历史学习的单词总数
- **连续天数** - 连续学习的天数（待实现）

### 3. 质量维度
- **正确率** - 所有答题的正确率
- **复习阶段** - 艾宾浩斯复习阶段（0-5）
- **下次复习** - 下次复习日期

```javascript

  const data = sessionStorage.getItem('studyResults')
```
